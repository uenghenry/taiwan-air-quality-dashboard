# Taiwan City Air Quality Dashboard

A simple Streamlit dashboard that collects current modeled air-quality data for representative locations in Taiwan.

## Files

- `app.py`: Streamlit application and data pipeline
- `requirements.txt`: Python dependencies
- `executive_summary.md`: one-page assignment summary
- `demo_script.md`: short presentation script

## Deployment

Deploy `app.py` through Streamlit Community Cloud using this GitHub repository.
